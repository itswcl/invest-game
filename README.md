# GeminiInvest - Investment Dashboard

## Features

- **Real-time Ticker Search**: Search for stocks like TSLA, AAPL, MSFT.
- **Interactive Price Charts**: Visualizing market trends with gradient area charts.
- **Portfolio Management**: Track your position and equity (Client-side demo).
- **Responsive Design**: Mobile-friendly layout with a premium dark mode aesthetic.
- **Mock Data Engine**: Generates realistic market data for demonstration purposes.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
npm run dev -- --turbopack
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This project is configured for automated deployment to **GitHub Pages** via GitHub Actions.

1. Push this code to a GitHub repository.
2. Go to Settings > Pages > Build and deployment > Source.
3. Select "GitHub Actions".
4. The deployment workflow will automatically run on push to `main`.
